# 二. 主要安全问题
* XSS (Cross Site Script): 跨站脚本注入
* CSRF (Cross Site Request Forge): 跨站请求伪造
* SQL注入 (SQL injection): SQL语言注入

<br />

# 三. 书籍

### (1) 白帽子讲Web安全
重点部分为:
* 浏览器安全 
* 跨站脚本攻击XSS
* 跨站请求伪造 (CSRF)
* 注入攻击
* 文件上传漏洞
* 认证和会话管理
* 访问控制
* Web 框架安全
* Web Server 配置安全
